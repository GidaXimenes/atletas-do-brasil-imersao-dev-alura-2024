## Desvendando Atletas - Imersão Dev com Google Gemini

**O que essa aplicação faz?**

Imagine que você é um super fã de esportes e quer saber tudo sobre seu atleta favorito ou sobre uma modalidade específica. Essa aplicação é a sua nova melhor amiga! Ela te ajuda a encontrar todas as informações que você precisa de forma rápida e fácil.

**Como ela funciona?**

1.  **Veja todos:** Assim que a página carrega, ela já exibe todos os atletas cadastrados.
2.  **Filtre por esporte:** Você pode usar os botões de filtro rápido (Skate, Judô, Tênis de Mesa, etc.) para ver apenas atletas de uma modalidade.
3.  **Digite e pesquise:** Você também pode digitar o nome do atleta ou esporte na caixa de pesquisa para encontrar resultados específicos.
4.  **Resultados na tela:** Se a aplicação encontrar algo, ela mostra pra você um card com a **foto** do atleta, seu nome, uma breve descrição e um link para saber mais. Se não encontrar nada, ela te avisa.

**Tecnologias usadas:**

- HTML: A estrutura básica da página, como os títulos, botões e onde os resultados vão aparecer.
- CSS: Deixa tudo bonitinho e organizado, com cores, fontes legais, imagens redondas e até uma barra de rolagem customizada.
- JavaScript: A parte mais inteligente da aplicação! É o JavaScript que faz a pesquisa, filtra por tags, compara os dados e mostra os resultados na tela.

**Como usar:**

1.  **Clone o repositório:** Faça um clone desse repositório no seu computador.
2.  **Abra o arquivo index.html:** Abra o arquivo `index.html` no seu navegador favorito.
3.  **Comece a pesquisar:** Use os filtros ou digite o nome do atleta e veja a mágica acontecer!

**Quer contribuir?**

Se você é um desenvolvedor e quer ajudar a melhorar essa aplicação, fique à vontade para abrir um pull request!

**Observações:**

- **Dados:** Os dados dos atletas estão no arquivo `dados.js`. Você pode adicionar mais dados aí se quiser (não esqueça de incluir a `imagem`!).
- **Design:** O design da página está no arquivo `style.css`. Se você quiser mudar as cores ou a fonte, é só editar esse arquivo.
- **Funcionalidades:** A lógica da pesquisa e dos filtros está no arquivo `app.js`. Se você quiser adicionar mais funcionalidades, é só editar esse arquivo.
